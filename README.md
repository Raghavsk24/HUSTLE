## HUSTLE

HUSTLE is a web application that recieves a file based input of the users resume which it can analyze using an ats scanner. The resume scanner comparator isolates keywords in the 
user's resume and asssigns a score between the user's resume and each side hsutle in our database. The side hustles recieving the highest score is then recommended to the user.

## Side Hustle Database
The functionality of our web application iss based on SQLite3 database that includes 16 side hustles, each of which can be filtered based on annual income and weekly comittment.
