## HUSTLE

HUSTLE is a web application that recieves a file based input of the user's resume which it can analyze using a unique resume scanner comparator system. The comparator isolates keywords in the user's resume and asssigns a score between the  resume and each side hustle in our database. The side hustle recieving the highest score is then recommended to the user.

## Side Hustle Database
The functionality of our web application is based on SQLite3 database that includes 16 side hustles, each of which can be filtered based on annual income and weekly comittment.
